# Diagrama de Caso de Uso

Os diagramas demonstram as utilidades de cada tela do jogo.

    Interface:
        (1) Jogar
        (2) Selecionar Nível

    Tabuleiro:
        (1) Movimentar as Peças
        (2) Rotacionar as Peças
        (3) Acelerar a Queda das Peças
        (4) Pausar o Jogo
        (5) Retomar ao Jogo
        (6) Sair do Jogo

    Menu Game Over:
        (1) Retornar para a Tela Inicial
        (2) Jogar Novamente
